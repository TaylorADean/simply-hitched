GIT instructions:

/1/ User set-up:
	$ git config --global user.name "Taylor Dean"
	$ git config --global user.email taylor.647@gmail.com
	
/2/ Default editor:
	$ git config --global core.editor "Path to notepad++"
	
/3/ Default compare tool:
	$ git config --global merge.tool "Path to winmerge"
	
/4/ Clone repo:
	$ git clone https://github.com/TaylorADean/simply-hitched.git

/5/ Set default push behaviour:
	$ git config --global push.default simple

/6/ First time push (set remote as upstream branch)
	$ git push --set-upstream origin master

/7/ Push
	$ git push

/8/ Fetch
	$ git fetch
